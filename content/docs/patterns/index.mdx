---
title: Patterns
description: Composições prontas — login, dashboard, pricing, onboarding.
---

Patterns são composições de múltiplos componentes que formam telas inteiras. Serão construídos na **Onda 7**, depois que os componentes individuais estiverem prontos.

Lista planejada:

- Login / Autenticação
- Dashboard com métricas
- Pricing Cards
- Onboarding / Setup Wizard
- Hero Sections
- Settings Form
- Empty States
- Modal de Ação Destrutiva
- Upgrade de Plano
