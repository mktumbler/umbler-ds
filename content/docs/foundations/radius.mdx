O Umbler DS usa cinco níveis de radius. **MD (10px)** é o padrão da maioria dos componentes de UI (inputs, botões, cards).

## Guideline de uso

| Token | Uso recomendado |
|---|---|
| `rounded-sm` | Tags inline, badges pequenas, indicadores |
| `rounded-md` | **Padrão** — botões, inputs, cards, modais |
| `rounded-lg` | Cards de destaque, hero containers |
| `rounded-xl` | Containers grandes, painéis de marketing |
| `rounded-full` | Avatares, pills, dots, círculos |
