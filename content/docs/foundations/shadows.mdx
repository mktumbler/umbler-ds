As sombras Umbler são adaptativas: no **dark mode** usam `neutral-900` com alta opacidade para cortar superfícies escuras sem "halo cinza"; no **light mode** usam `neutral-500` (índigo-ardósia) com opacidade reduzida para um resultado sutil e com identidade de marca.

## Quando usar cada um

| Token | Uso |
|---|---|
| `shadow-xs` | Hover sutil em itens de lista |
| `shadow-sm` | Cards estáticos, botões em repouso |
| `shadow-md` | Dropdowns, popovers, tooltips |
| `shadow-lg` | Modals, sheets, command palette |
| `shadow-xl` | Elementos elevados máximos (alert dialogs) |
| `shadow-glow` | Estados de foco, CTAs primários, destaque de marca |
