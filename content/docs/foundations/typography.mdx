A tipografia Umbler usa **P22 Mackinac** para títulos — uma serifada com personalidade editorial — **Inter** para textos de UI e corpo, e **JetBrains Mono** para código.

## Famílias

| Família | Uso | Token |
|---|---|---|
| P22 Mackinac | Títulos (Display, H1–H4) | `--font-heading` |
| Inter | UI, body, labels | `--font-sans` |
| JetBrains Mono | Código, terminais, dados estruturados | `--font-mono` |

## Tokens

Cada step da escala tem um token semântico Tailwind v4:

```css
text-display   /* 72px / 700 */
text-h1        /* 48px / 700 */
text-h2        /* 36px / 600 */
text-h3        /* 28px / 600 */
text-h4        /* 22px / 600 */
text-body-lg   /* 18px / 400 */
text-body      /* 16px / 400 */
text-body-sm   /* 14px / 400 */
text-caption   /* 12px / 500 */
```
