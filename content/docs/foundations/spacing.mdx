Todo espaçamento no Umbler DS deriva de múltiplos de **4px**. Isso garante alinhamento visual consistente em qualquer composição.

## Grid & Layout

| Token | Valor | Uso |
|---|---|---|
| `--container-max` | 1200px | Largura máxima do container principal |
| `--grid-gutter` | 24px | Espaço entre colunas |
| `--grid-margin` | 32px | Margem externa do grid |

## Breakpoints

| Nome | Min-width | Uso típico |
|---|---|---|
| xs | 0px | Mobile portrait |
| sm | 640px | Mobile landscape |
| md | 768px | Tablet |
| lg | 1024px | Desktop (sidebar visível) |
| xl | 1280px | Wide desktop |

## Como usar

Tailwind v4 expõe a escala automaticamente:

```tsx
<div className="p-4 gap-6 mt-12" />
// p-4  → 16px padding
// gap-6 → 24px gap
// mt-12 → 48px margin-top
```

Para valores customizados com tokens semânticos, use CSS variables:

```tsx
<div style={{ padding: 'var(--spacing-6)' }} />
```
