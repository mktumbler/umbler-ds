### Stagger (delay base para sequências)

Quando vários itens entram em série (cards de feature, linhas de tabela, palavras de um hero), use stagger como **delay incremental por índice**. Cada item começa após `stagger × index`.

## Princípios

1. **Nunca acima de 600ms** em UI funcional do produto. `--duration-slowest` é reservado pra landing/marketing — hero reveals, manifesto, splashes onde o tempo de tela é mais longo.
2. **Sempre `ease-out` para entradas** — sensação de chegada natural.
3. **Use `prefers-reduced-motion`** para respeitar acessibilidade:

```css
@media (prefers-reduced-motion: reduce) {
  * {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
  }
}
```
