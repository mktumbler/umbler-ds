Transacional de onboarding: confirma a posse do endereço com CTA direto e um **código
numérico alternativo** (para quem abre o e-mail num dispositivo e digita no outro).

## Uso

```tsx
import { render } from '@react-email/render';
import ConfirmationEmail from '@/emails/templates/confirmation';

const html = await render(
  <ConfirmationEmail
    firstName="Ana"
    confirmUrl={`https://app.umbler.com/confirm?token=${token}`}
    code="482 916"
    expiresIn="24 horas"
  />
);
```

## Estrutura do template

- **Badge** — "Quase lá" em verde (estado positivo, não ação)
- **Título** + parágrafo curto
- **CTA** — "Confirmar e-mail"
- **Card de código** — código numérico em mono, centralizado, pra fluxo cross-device
- **Expiração** + nota "não criou conta? ignore"
