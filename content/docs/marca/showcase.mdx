## O que ainda não está

Iteração futura — cada uma vira commit separado quando entrar:

- **Problem/Solution** (pares dor → resposta)
- **Steps 01/02/03** (como ativar)
- **Manifesto** (pico emocional pré-pricing)
- **Pricing Table** (toggle anual + plano destacado)
- **Testimonial** (depoimentos específicos)
- **FAQ** (objeções desarmadas)
- **Wow moments** (max 2-3 — possivelmente shine border em CTA, gradient text no manifesto)

## Por que showcase importa

Doc de DS sem prova viva sofre o **"funcionará na minha LP?"** silencioso. O showcase responde:

- **Visualmente** — quem chega vê o resultado, não slide
- **Tecnicamente** — código está aqui no repo, copiável
- **Em produção** — rota real, build real, deploy real

Quando um time avalia se adota o Umbler DS, manda o link do showcase. Se a landing convence, o DS vendeu sozinho.
