A landing Umbler **não é institucional decorativo**. Existe para converter, explicar com clareza, dar contexto para dezenas de páginas e funcionar como máquina de aquisição. Esta seção é o **playbook** que orienta como cada peça da LP deve operar.

Brand define **como soa**. Conversion define **como atua**.

## Premissa

A LP existe para **converter**, não para impressionar. Cada seção tem um propósito persuasivo. Quem não tem, sai. Visitante decide em 5 segundos se vai continuar; o resto da página existe para ratificar essa primeira decisão e levar ao CTA.
