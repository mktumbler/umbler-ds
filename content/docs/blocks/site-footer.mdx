## API

```tsx
import { SiteFooter } from '@/components/blocks/site-footer';
import { UmblerLogo } from '@/components/ui/umbler-logo';
import { LinkedinLogo } from '@phosphor-icons/react/dist/ssr';

<SiteFooter
  logo={<UmblerLogo className="h-7 w-auto text-white" />}
  tagline="A infraestrutura por trás dos negócios que crescem."
  columns={[
    { title: 'Produto', links: [{ label: 'Talk', href: '/talk' }] },
    { title: 'Empresa', links: [{ label: 'Sobre', href: '/about' }] },
  ]}
  social={[
    { label: 'LinkedIn', href: '#', icon: <LinkedinLogo size={16} weight="fill" /> },
  ]}
  copyright="© 2026 Umbler. Todos os direitos reservados."
  legal={[{ label: 'Privacidade', href: '/privacy' }]}
/>
```

## Boas práticas

- **3-4 colunas funcionam melhor** — abaixo disso fica vazio, acima fica denso no mobile
- **Tagline curta** — uma linha, voz da marca, não slogan publicitário
- **Não duplique o header** — footer é navegação secundária; só inclua o que o header não cobre
- **Copyright sempre atual** — gere a partir de `new Date().getFullYear()` no consumer, não hardcode
