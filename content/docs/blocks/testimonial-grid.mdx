Dois layouts para exibir múltiplos depoimentos: `wall` empilha cards em 3 colunas, `featured` destaca um depoimento longo ao lado de menores.

---

## Wall of Love

3 colunas de cards sem borda. Para 3 a 12 depoimentos — distribui automaticamente entre as colunas.

## API

```tsx
import { TestimonialGrid, type TestimonialItem } from '@/components/blocks/testimonial-grid';

const testimonials: TestimonialItem[] = [
  {
    quote: 'Em 30 dias o tempo de resposta caiu pela metade.',
    author: 'Rodrigo Menezes',
    role: 'Gerente Comercial',
    company: 'Clínica Vida Plena',
    avatarSrc: '/images/rodrigo.jpg', // opcional — mostra iniciais se omitido
    stars: 5,
  },
  // ...
];

<TestimonialGrid
  eyebrow="Depoimentos"
  headline="Quem usa, recomenda"
  subheadline="Mais de 15 mil empresas usam o Umbler Talk."
  testimonials={testimonials}
  variant="wall"    // 'wall' | 'featured'
  className="bg-surface"
/>
```

### TestimonialItem

| Prop | Tipo | Descrição |
|---|---|---|
| `quote` | `string` | Texto do depoimento |
| `author` | `string` | Nome do autor |
| `role` | `string` | Cargo |
| `company` | `string?` | Empresa (exibida após `·`) |
| `avatarSrc` | `string?` | URL da foto — usa iniciais se omitido |
| `stars` | `1–5?` | Estrelas (omitido = sem estrelas) |

### TestimonialGrid props

| Prop | Tipo | Padrão |
|---|---|---|
| `testimonials` | `TestimonialItem[]` | obrigatório |
| `variant` | `'wall' \| 'featured'` | `'wall'` |
| `eyebrow` | `string?` | — |
| `headline` | `string?` | — |
| `subheadline` | `string?` | — |
| `className` | `string?` | — |
