## API

```tsx
import { LogoCloud } from '@/components/blocks/logo-cloud';

<LogoCloud
  variant="marquee"
  speed={25}
  label="+60 mil empresas confiam na Umbler"
  logos={[
    { src: '/logos/unimed.svg', alt: 'Unimed' },
    { src: '/logos/cora.svg', alt: 'Cora', maxHeight: 28 },
  ]}
/>
```

## Boas práticas

- **SVG sempre que possível** — nítido em qualquer tamanho e responde bem ao grayscale
- Grid funciona melhor com 4, 6 ou 8 logos (a grade se adapta); marquee brilha com 6+
- Logos precisam de cor sólida com contraste — efeitos sutis somem em grayscale + opacity
- Acompanhe de um número âncora no `label` ("+60 mil empresas") — logo sem contexto prova pouco
