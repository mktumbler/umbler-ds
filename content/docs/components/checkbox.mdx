## API

```tsx
import { Checkbox } from '@/components/ui/checkbox';

// Simples
<Checkbox label="Aceitar os termos de uso" />

// Com description e estado de erro
<Checkbox
  state="error"
  label="Aceitar os termos"
  description="Obrigatório para continuar."
/>

// Grupo com fieldset
<fieldset>
  <legend>Notificações</legend>
  <Checkbox label="E-mails de marketing" />
  <Checkbox label="Alertas de segurança" defaultChecked />
</fieldset>
```

| Prop          | Tipo                      | Padrão       | Descrição                              |
| ------------- | ------------------------- | ------------ | -------------------------------------- |
| `label`       | `string`                  | —            | Texto principal ao lado do checkbox.  |
| `description` | `string`                  | —            | Texto auxiliar abaixo do label.       |
| `state`       | `'default' \| 'error'`    | `'default'`  | Estado de validação visual.           |
| `disabled`    | `boolean`                 | `false`      | Desabilita o checkbox.                |
| `...input`    | `InputHTMLAttributes`     | —            | Todos os atributos nativos de input.  |
