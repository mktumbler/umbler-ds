Checkbox usa a API nativa `<input type="checkbox">` com `appearance-none` e classes `peer-*` do Tailwind para a aparência personalizada. Suporta os estados `checked`, `indeterminate` e `disabled` sem JavaScript extra (exceto `indeterminate`, que requer uma ref no cliente).

## Composição

```tsx
import { Checkbox } from '@/components/ui/checkbox';

// Simples
<Checkbox label="Aceito os termos de uso" />

// Com description e estado de erro
<Checkbox
  state="error"
  label="Aceito os termos de uso"
  description="Você precisa aceitar os termos para continuar."
/>

// Indeterminate (client component)
'use client';
import { useEffect, useRef } from 'react';

function IndeterminateExample() {
  const ref = useRef<HTMLInputElement>(null);
  useEffect(() => {
    if (ref.current) ref.current.indeterminate = true;
  }, []);
  return <input ref={ref} type="checkbox" />;
}
```
