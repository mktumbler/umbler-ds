## API

| Prop           | Tipo                    | Default | Descrição                                      |
| -------------- | ----------------------- | ------- | ---------------------------------------------- |
| `page`         | `number`                | —       | Página ativa (1-indexed).                      |
| `totalPages`   | `number`                | —       | Total de páginas. Se ≤ 1, não renderiza.       |
| `onPageChange` | `(n: number) => void`   | —       | Disparado quando o usuário troca de página.    |
| `siblings`     | `number`                | `1`     | Quantas páginas visíveis ao redor da atual.    |

## Acessibilidade

- `<nav aria-label="Paginação">` agrupa.
- Página atual marcada com `aria-current="page"`.
- Botões prev/next têm `aria-label` e ficam `disabled` nos extremos.

## Uso

```tsx
'use client';
import { useState } from 'react';
import { Pagination } from '@/components/ui/pagination';

const [page, setPage] = useState(1);

<Pagination page={page} totalPages={20} onPageChange={setPage} />
```
