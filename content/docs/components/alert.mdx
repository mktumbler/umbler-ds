## API

```tsx
import { Alert } from '@/components/ui/alert';

// Simples
<Alert variant="success">Deploy concluído com sucesso.</Alert>

// Com título e dismiss
<Alert
  variant="warning"
  title="Domínio expirando"
  onClose={() => setVisible(false)}
>
  Seu domínio <strong>umbler.com</strong> expira em 7 dias.
</Alert>
```

| Prop      | Tipo                                            | Default  | Descrição                          |
| --------- | ----------------------------------------------- | -------- | ---------------------------------- |
| `variant` | `'info' \| 'success' \| 'warning' \| 'error'`  | `'info'` | Cor e ícone semântico              |
| `title`   | `string`                                        | —        | Cabeçalho em negrito               |
| `icon`    | `boolean`                                       | `true`   | Exibe ou oculta o ícone            |
| `onClose` | `() => void`                                    | —        | Callback do botão fechar           |
| `...div`  | `HTMLAttributes<HTMLDivElement>`                | —        | Atributos nativos de div           |
