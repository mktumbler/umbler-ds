O `TagInput` é um campo controlado para entrada de múltiplas tags (palavras-chave, destinatários, filtros). Reúsa o componente `Tag` para os chips e é dimensionado **igual ao `Input`** (mesmos `size` e `state`), então encaixa lado-a-lado em formulários.

## Básico

<ComponentPreview title="TagInput">
  <TagInputDemo />
</ComponentPreview>

Enter ou vírgula adiciona a tag; Backspace com o campo vazio remove a última. O campo cresce em altura quando as tags quebram para a próxima linha.

## Paridade com o Input

Na primeira linha, o `TagInput md` tem a mesma altura do `Input md` (40px) — pode alinhar os dois no mesmo formulário.

## Uso

```tsx
'use client';
import { useState } from 'react';
import { TagInput } from '@/components/ui/tag-input';

function Form() {
  const [tags, setTags] = useState<string[]>(['Next.js']);
  return <TagInput value={tags} onChange={setTags} placeholder="Adicionar tag…" />;
}
```
