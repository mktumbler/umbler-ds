O `TagInput` é um campo controlado para entrada de múltiplas tags (palavras-chave, destinatários, filtros). Reusa o componente `Tag` para os chips e é dimensionado **igual ao `Input`** (mesmos `size` e `state`), então encaixa lado-a-lado em formulários.

## API

| Prop          | Tipo                                                         | Default          | Descrição                           |
| ------------- | ------------------------------------------------------------ | ---------------- | ----------------------------------- |
| `value`       | `string[]`                                                   | —                | Tags atuais (controlado)            |
| `onChange`    | `(tags: string[]) => void`                                   | —                | Disparado quando a lista muda       |
| `size`        | `'sm' \| 'md' \| 'lg'`                                      | `'md'`           | Altura, igual ao Input              |
| `state`       | `'default' \| 'error' \| 'success'`                         | `'default'`      | Estado de validação (borda + ring)  |
| `tagVariant`  | `'brand' \| 'success' \| 'warning' \| 'error' \| 'neutral'`| `'brand'`        | Cor das tags                        |
| `placeholder` | `string`                                                     | `'Adicionar…'`   | Texto quando não há tags            |
| `dedupe`      | `boolean`                                                    | `true`           | Impede tags duplicadas              |
| `disabled`    | `boolean`                                                    | `false`          | Desabilita o campo                  |
| `className`   | `string`                                                     | —                | Classes extras                      |

## Uso

```tsx
'use client';
import { useState } from 'react';
import { TagInput } from '@/components/ui/tag-input';

function Form() {
  const [tags, setTags] = useState<string[]>(['Next.js']);
  return <TagInput value={tags} onChange={setTags} placeholder="Adicionar tag…" />;
}
```
