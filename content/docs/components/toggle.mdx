Toggle renderiza um `<button role="switch">` semântico — não um input — com track e thumb animados via Tailwind. Suporta modo **controlled** (`checked` + `onChange`) e **uncontrolled** (`defaultChecked`).

## Composição

```tsx
import { Toggle } from '@/components/ui/toggle';

// Uncontrolled
<Toggle defaultChecked label="Notificações" description="Receba alertas por e-mail." />

// Controlled
const [enabled, setEnabled] = useState(false);
<Toggle checked={enabled} onChange={setEnabled} label="Tema escuro" />
```
