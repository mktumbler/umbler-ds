## API

```tsx
import { Toggle } from '@/components/ui/toggle';

// Uncontrolled
<Toggle
  label="Notificações por e-mail"
  description="Alertas e resumos diretamente no e-mail."
  defaultChecked
/>

// Controlled
<Toggle
  checked={isEnabled}
  onChange={(e) => setEnabled(e.target.checked)}
  size="sm"
/>
```

| Prop          | Tipo                         | Padrão  | Descrição                              |
| ------------- | ---------------------------- | ------- | -------------------------------------- |
| `size`        | `'sm' \| 'md'`               | `'md'`  | Tamanho do switch.                    |
| `label`       | `string`                     | —       | Texto à direita do switch.            |
| `description` | `string`                     | —       | Texto auxiliar abaixo do label.       |
| `disabled`    | `boolean`                    | `false` | Desabilita o toggle.                  |
| `...input`    | `InputHTMLAttributes`        | —       | Atributos nativos de input checkbox.  |
