## API

```tsx
import { Select } from '@/components/ui/select';
import { InputGroup, InputLabel, InputHint } from '@/components/ui/input';

<InputGroup>
  <InputLabel htmlFor="framework">Framework</InputLabel>
  <Select id="framework" defaultValue="next">
    <option value="next">Next.js</option>
    <option value="nuxt">Nuxt</option>
    <option value="astro">Astro</option>
  </Select>
</InputGroup>
```

| Prop       | Tipo                                | Padrão      | Descrição                             |
| ---------- | ----------------------------------- | ----------- | ------------------------------------- |
| `state`    | `'default' \| 'error' \| 'success'` | `'default'` | Estado de validação visual.          |
| `size`     | `'sm' \| 'md' \| 'lg'`             | `'md'`      | Altura do campo.                     |
| `disabled` | `boolean`                           | `false`     | Desabilita o select.                 |
| `...select`| `SelectHTMLAttributes`              | —           | Atributos nativos de select.         |
