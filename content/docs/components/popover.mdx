Diferente do `Tooltip` (texto curto, abre no hover) e do `Dropdown` (menu de ações navegável por setas), o `Popover` é um **container livre** clicável por dentro. Use para mini-formulários, picks, ações secundárias com explicação.

## Com formulário

Aceita qualquer ReactNode — inputs, botões, conteúdo rico. Use `PopoverClose` para fechar via ação interna.

## Com arrow + ícone trigger

A prop `arrow` adiciona uma setinha apontando pro trigger — útil quando o trigger é um ícone discreto.

## Acessibilidade

- Trigger ganha `aria-expanded` e `aria-controls` automaticamente.
- Foco move pro conteúdo quando abre; volta pro trigger quando fecha.
- Fecha em `Escape` ou clique fora.
- Para trigger sem texto visível (ícone), forneça `aria-label` no botão.

## Uso

```tsx
import {
  Popover,
  PopoverContent,
  PopoverTrigger,
  PopoverClose,
} from '@/components/ui/popover';

<Popover>
  <PopoverTrigger asChild>
    <Button>Abrir</Button>
  </PopoverTrigger>
  <PopoverContent>
    <p>Conteúdo do popover</p>
    <PopoverClose asChild>
      <Button size="sm">Fechar</Button>
    </PopoverClose>
  </PopoverContent>
</Popover>
```
