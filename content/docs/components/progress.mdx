## API — Progress

| Prop    | Tipo                                            | Default   | Descrição                          |
| ------- | ----------------------------------------------- | --------- | ---------------------------------- |
| `value` | `number \| null`                                | —         | 0–`max`. Omita para indeterminado. |
| `max`   | `number`                                        | `100`     | Valor máximo.                      |
| `size`  | `'sm' \| 'md' \| 'lg'`                         | `'md'`    | Espessura da barra.                |
| `tone`  | `'brand' \| 'success' \| 'warning' \| 'error'` | `'brand'` | Cor do indicador.                  |

## API — ProgressCircle

Mesmas props `value` / `max` / `tone`. `size` muda o diâmetro (28 / 40 / 56px). `showLabel` exibe o percentual no centro.

## Acessibilidade

- Radix Progress aplica `role="progressbar"` + `aria-valuenow`/`min`/`max` automaticamente.
- `ProgressCircle` aplica os mesmos atributos manualmente via SVG ARIA.
- No modo indeterminado, `aria-valuenow` é omitido — leitores anunciam "carregando".

## Uso

```tsx
import { Progress, ProgressCircle } from '@/components/ui/progress';

// Determinado
<Progress value={uploaded} max={total} />

// Indeterminado
<Progress />

// Circular com label
<ProgressCircle value={64} showLabel size="lg" />
```
