O `Avatar` exibe a identidade do usuário em três níveis de fallback: foto → iniciais → ícone. Use `AvatarGroup` para representar múltiplos participantes de forma compacta.

## Fallbacks

O Avatar usa a foto quando disponível. Se `src` falhar ou não existir, gera iniciais a partir de `name`. Sem `name`, exibe um ícone genérico.

## Grupo

`AvatarGroup` exibe avatars sobrepostos com um overflow `+N` quando a lista ultrapassa `max`.

## API — Avatar

| Prop        | Tipo                                     | Default | Descrição                           |
| ----------- | ---------------------------------------- | ------- | ----------------------------------- |
| `src`       | `string`                                 | —       | URL da imagem. Ao falhar, usa fallback |
| `alt`       | `string`                                 | —       | Texto alternativo da imagem         |
| `name`      | `string`                                 | —       | Nome completo — gera até 2 iniciais |
| `size`      | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl'` | `'md'`  | Tamanho: 24 / 32 / 40 / 48 / 64px  |
| `className` | `string`                                 | —       | Classes extras                      |

## API — AvatarGroup

| Prop        | Tipo              | Default | Descrição                                   |
| ----------- | ----------------- | ------- | ------------------------------------------- |
| `avatars`   | `AvatarProps[]`   | —       | Lista de avatars                            |
| `max`       | `number`          | —       | Máximo visível; o resto vira `+N`           |
| `size`      | `AvatarProps["size"]` | `'md'` | Tamanho aplicado a todos no grupo       |
| `className` | `string`          | —       | Classes extras no wrapper                   |

## Uso

```tsx
import { Avatar, AvatarGroup } from '@/components/ui/avatar';

// Com foto
<Avatar src="/foto.jpg" name="Ana Lima" size="md" />

// Apenas iniciais
<Avatar name="Bruno Costa" size="lg" />

// Grupo com overflow
<AvatarGroup
  avatars={[
    { name: 'Ana Lima', src: '/a.jpg' },
    { name: 'Bruno Costa' },
    { name: 'Carla Dias' },
    { name: 'Diego Melo' },
  ]}
  max={3}
  size="sm"
/>
```
