O `Avatar` exibe a identidade do usuário em três níveis de fallback: foto → iniciais → ícone. Use `AvatarGroup` para representar múltiplos participantes de forma compacta.

## Uso

```tsx
import { Avatar, AvatarGroup } from '@/components/ui/avatar';

// Com foto
<Avatar src="/foto.jpg" name="Ana Lima" size="md" />

// Apenas iniciais
<Avatar name="Bruno Costa" size="lg" />

// Sem dados (ícone)
<Avatar size="sm" />

// Grupo com overflow
<AvatarGroup
  avatars={[
    { name: 'Ana Lima', src: '/a.jpg' },
    { name: 'Bruno Costa' },
    { name: 'Carla Dias' },
    { name: 'Diego Melo' },
  ]}
  max={3}
  size="sm"
/>
```
