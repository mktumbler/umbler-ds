O `Spinner` é um arco giratório que usa `currentColor`, adaptando-se a qualquer contexto pelo `text-*` do pai — botões, overlays, áreas de loading.

## Uso

```tsx
import { Spinner } from '@/components/ui/spinner';

// Standalone
<Spinner size="lg" className="text-brand-400" />

// Inline em botão
<button disabled>
  <Spinner size="sm" />
  Salvando...
</button>
```
