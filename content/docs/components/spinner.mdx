O `Spinner` é um arco giratório que usa `currentColor`, adaptando-se a qualquer contexto pelo `text-*` do pai — botões, overlays, áreas de loading.

## Tamanhos

<VariantGrid columns={1}>
  <VariantCard label="sm / md / lg / xl" hint="14 / 16 / 20 / 24px — controle via prop size">
    <SpinnerSizes />
  </VariantCard>
</VariantGrid>

## Cores

Controle via `className` com qualquer classe `text-*`. O SVG usa `currentColor`.

<VariantGrid columns={1}>
  <VariantCard label="brand / success / error / muted" hint="text-brand-400, text-success-500, text-error-500, text-foreground-muted">
    <SpinnerColors />
  </VariantCard>
</VariantGrid>

## API

| Prop        | Tipo                           | Default | Descrição                         |
| ----------- | ------------------------------ | ------- | --------------------------------- |
| `size`      | `'sm' \| 'md' \| 'lg' \| 'xl'`| `'md'`  | Tamanho: 14 / 16 / 20 / 24px     |
| `className` | `string`                       | —       | Classes extras — `text-*` para cor |

## Uso

```tsx
import { Spinner } from '@/components/ui/spinner';

// Standalone
<Spinner size="lg" className="text-brand-400" />

// Inline em botão
<button disabled>
  <Spinner size="sm" />
  Salvando...
</button>
```
