O `Spinner` é usado para indicar estados de carregamento. Herda a cor do elemento pai via `currentColor`, tornando-o facilmente adaptável a qualquer contexto.

## Cores

A cor é controlada pela propriedade `color` do elemento pai (ex.: `text-brand-300`).

## Uso

```tsx
import { Spinner } from '@/components/ui/spinner';

// Padrão
<Spinner />

// Tamanho grande, cor de erro
<Spinner size="lg" className="text-error-500" />

// Em um botão customizado
<button className="text-brand-300">
  <Spinner size="sm" />
  Carregando...
</button>
```
