Mesmo paradigma do [`Dialog`](/docs/components/dialog), mas o conteúdo entra deslizando de uma das bordas em vez de aparecer centralizado. Útil para painéis de detalhe, drawers de notificação, menus mobile e formulários laterais.

Construído sobre `@radix-ui/react-dialog` — foco trap, `Esc` para fechar, scroll lock e `aria-*` corretos.

## API — SheetContent

| Prop        | Tipo                                              | Default   | Descrição                                                        |
| ----------- | ------------------------------------------------- | --------- | ---------------------------------------------------------------- |
| `side`      | `'right' \| 'left' \| 'top' \| 'bottom'`         | `'right'` | Borda de onde o painel desliza.                                  |
| `showClose` | `boolean`                                         | `true`    | Exibe o × no canto superior direito.                             |
| `className` | `string`                                          | —         | Default: `sm:max-w-md` (right/left) ou `max-h-[85vh]` (top/bottom). |

## Slots

| Componente           | Função                                                       |
| -------------------- | ------------------------------------------------------------ |
| `<SheetHeader>`      | Título + descrição, padding reservado para o ×               |
| `<SheetTitle>`       | Título acessível (`aria-labelledby` automático)              |
| `<SheetDescription>` | Texto auxiliar (`aria-describedby` automático)               |
| `<SheetBody>`        | Área rolável entre header e footer (`overflow-y-auto`)       |
| `<SheetFooter>`      | Ações sticky no rodapé com borda separadora                  |
| `<SheetClose>`       | Wrapper Radix que fecha o sheet (use com `asChild`)          |

## Acessibilidade

- Foco trap automático — `Tab` circula só dentro do sheet.
- `Esc` fecha; clicar no overlay fecha.
- Sempre forneça `<SheetTitle>` (mesmo com `sr-only`) — Radix avisa no console se faltar.
- O × de fechar tem `aria-label="Fechar"`.

## Uso

```tsx
import {
  Sheet, SheetTrigger, SheetContent, SheetHeader,
  SheetTitle, SheetDescription, SheetBody, SheetFooter, SheetClose,
} from '@/components/ui/sheet';

<Sheet>
  <SheetTrigger asChild>
    <Button>Abrir detalhes</Button>
  </SheetTrigger>
  <SheetContent side="right">
    <SheetHeader>
      <SheetTitle>Detalhes</SheetTitle>
      <SheetDescription>Resumo da conversa.</SheetDescription>
    </SheetHeader>
    <SheetBody>
      {/* conteúdo rolável */}
    </SheetBody>
    <SheetFooter>
      <SheetClose asChild>
        <Button variant="secondary">Fechar</Button>
      </SheetClose>
      <Button>Salvar</Button>
    </SheetFooter>
  </SheetContent>
</Sheet>
```
