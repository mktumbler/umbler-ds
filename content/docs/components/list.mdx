Compound semântico em cima de `<ul>` / `<li>`, pensado para o "list group" tradicional — pastas, conversas, configurações, atividades. Para tabelas com colunas e seleção use [`Table`](/docs/components/table); para feed de eventos no tempo use [`Timeline`](/docs/components/timeline).

## API — List

| Prop          | Tipo      | Default | Descrição                                              |
| ------------- | --------- | ------- | ------------------------------------------------------ |
| `bordered`    | `boolean` | `false` | Borda externa + cantos arredondados.                   |
| `dividers`    | `boolean` | `false` | Separador entre os itens.                              |
| `dense`       | `boolean` | `false` | Padding reduzido.                                      |
| `interactive` | `boolean` | `false` | Aplica hover/cursor em todos os itens via Context.     |

## API — ListItem

| Prop          | Tipo      | Default  | Descrição                                                                    |
| ------------- | --------- | -------- | ---------------------------------------------------------------------------- |
| `interactive` | `boolean` | herda    | Sobrescreve o `interactive` do `<List>` para este item.                      |
| `onClick`     | `fn`      | —        | Quando presente, o item ganha `role="button"` e `tabIndex={0}`.              |

## Slots

| Componente            | Função                                         |
| --------------------- | ---------------------------------------------- |
| `<ListItemLeading>`   | Ícone ou avatar à esquerda                     |
| `<ListItemContent>`   | Bloco central (título + descrição), flex-1     |
| `<ListItemTitle>`     | Título truncado com text-overflow: ellipsis    |
| `<ListItemDescription>` | Texto secundário                             |
| `<ListItemTrailing>`  | Ação, badge ou timestamp à direita             |

## Uso

```tsx
import {
  List, ListItem, ListItemLeading, ListItemContent,
  ListItemTitle, ListItemDescription, ListItemTrailing,
} from '@/components/ui/list';

<List bordered dividers interactive>
  <ListItem onClick={() => openConversation(c.id)}>
    <ListItemLeading><Avatar size="sm" name={c.name} /></ListItemLeading>
    <ListItemContent>
      <ListItemTitle>{c.name}</ListItemTitle>
      <ListItemDescription>{c.lastMessage}</ListItemDescription>
    </ListItemContent>
    <ListItemTrailing>
      {c.unread > 0 && <Badge variant="brand">{c.unread}</Badge>}
    </ListItemTrailing>
  </ListItem>
</List>
```
