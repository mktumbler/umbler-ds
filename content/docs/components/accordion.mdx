Construído sobre Radix Accordion — gerencia foco, navegação por teclado e estado. A animação de altura aproveita `--radix-accordion-content-height` exposta pelo Radix.

## Quando usar cada `type`

| `type`     | Quando                                                                    |
| ---------- | ------------------------------------------------------------------------- |
| `single`   | FAQ, conteúdo mutuamente exclusivo — uma aberta não ajuda entender outra  |
| `multiple` | Configurações, checklist — usuário pode comparar seções abertas           |

## Acessibilidade

- Trigger é um `<button>` dentro de `<h3>` (Radix Header) — hierarquia semântica correta.
- ↑↓ navegam entre triggers; `Home`/`End` vão para o primeiro/último.
- `Space`/`Enter` abrem/fecham a seção focada.

## Uso

```tsx
import {
  Accordion,
  AccordionContent,
  AccordionItem,
  AccordionTrigger,
} from '@/components/ui/accordion';

// Single collapsible
<Accordion type="single" collapsible defaultValue="item-1">
  <AccordionItem value="item-1">
    <AccordionTrigger>Pergunta 1</AccordionTrigger>
    <AccordionContent>Resposta 1</AccordionContent>
  </AccordionItem>
  <AccordionItem value="item-2">
    <AccordionTrigger>Pergunta 2</AccordionTrigger>
    <AccordionContent>Resposta 2</AccordionContent>
  </AccordionItem>
</Accordion>

// Multiple
<Accordion type="multiple" defaultValue={['item-1', 'item-2']}>
  ...
</Accordion>
```
