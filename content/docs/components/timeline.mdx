Compound semântico em cima de `<ol>`/`<li>`, com um indicador (dot) por evento e uma linha conectora vertical. Cobre dois usos principais: **feed de atividade** (com ícones e texto rico) e **changelog compacto** (versão + data + descrição curta).

## API — Timeline

| Prop      | Tipo      | Default | Descrição                                          |
| --------- | --------- | ------- | -------------------------------------------------- |
| `compact` | `boolean` | `false` | Reduz o espaçamento entre os itens via Context.    |

## API — TimelineDot

| Prop     | Tipo                                                         | Default     | Descrição                                      |
| -------- | ------------------------------------------------------------ | ----------- | ---------------------------------------------- |
| `tone`   | `'default' \| 'brand' \| 'success' \| 'warning' \| 'error'` | `'default'` | Cor do anel (e do miolo / ícone).              |
| `filled` | `boolean`                                                    | `true`      | Pinta o miolo. `false` → só anel (pendente).   |
| `icon`   | `ReactNode`                                                  | —           | Ícone Phosphor — gera dot maior (24px).        |

## Acessibilidade

- `<Timeline>` renderiza `<ol>` — leitores anunciam a ordem dos eventos.
- O dot é `aria-hidden="true"` (decorativo).
- Use `<TimelineTime dateTime="...">` com o timestamp ISO para SEO e leitura por máquinas.

## Uso

```tsx
import {
  Timeline, TimelineItem, TimelineDot,
  TimelineContent, TimelineTitle, TimelineTime,
} from '@/components/ui/timeline';
import { Rocket } from '@phosphor-icons/react';

<Timeline>
  <TimelineItem>
    <TimelineDot tone="brand" icon={<Rocket weight="fill" />} />
    <TimelineContent>
      <TimelineTime dateTime="2026-05-28T14:02:00Z">há 5 min</TimelineTime>
      <TimelineTitle>Deploy em produção</TimelineTitle>
      <p>Versão v2.14.0 publicada.</p>
    </TimelineContent>
  </TimelineItem>
</Timeline>
```
