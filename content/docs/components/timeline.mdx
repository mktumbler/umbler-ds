Compound semântico em cima de `<ol>`/`<li>`, com um indicador (dot) por evento e uma linha conectora vertical desenhada via pseudo-elemento. Cobre dois usos principais: **feed de atividade** (com ícones e texto rico) e **changelog compacto** (versão + data + descrição curta).

## Activity feed

Dot maior com ícone Phosphor, título, timestamp relativo e descrição. Tons semânticos (`brand`, `success`, `warning`, `error`) destacam o tipo do evento.

<ComponentPreview title="Log de deploys e mudanças">

</ComponentPreview>

## Changelog compacto

`compact` aperta o espaçamento entre itens — bom pra mostrar muitos releases numa coluna estreita. Combina bem com `Badge` para marcar `NEW`/`FIX`/`BREAKING`.

</ComponentPreview>

## Minimal

Sem ícones, só o dot. Útil pra steps lineares (status de pedido, onboarding). O último dot com `filled={false}` indica "pendente".

## Props

### `<Timeline>`

| Prop | Tipo | Default | Descrição |
|---|---|---|---|
| `compact` | `boolean` | `false` | Reduz o espaçamento entre os itens. Propagado via Context. |

### `<TimelineDot>`

| Prop | Tipo | Default | Descrição |
|---|---|---|---|
| `tone` | `'default' \| 'brand' \| 'success' \| 'warning' \| 'error'` | `'default'` | Cor do anel (e do miolo / ícone). |
| `filled` | `boolean` | `true` | Pinta o miolo do dot. `false` deixa só o anel — bom pra steps pendentes. |
| `icon` | `ReactNode` | — | Ícone (Phosphor recomendado). Vira automaticamente um dot maior (24px). |

## Acessibilidade

- `<Timeline>` renderiza `<ol>` — leitores anunciam a ordem dos eventos.
- O dot é `aria-hidden="true"` (decorativo) — toda a informação vive no conteúdo.
- Use `<TimelineTime>` com `dateTime="2026-05-28T14:02:00Z"` quando tiver o timestamp absoluto, mesmo que exiba relativo: melhora SEO e leitura por máquinas.

```tsx
<TimelineTime dateTime="2026-05-28T14:02:00Z">há 5 min</TimelineTime>
```

## Uso

```tsx
import {
  Timeline,
  TimelineItem,
  TimelineDot,
  TimelineContent,
  TimelineTitle,
  TimelineTime,
} from '@/components/ui/timeline';
import { Rocket } from '@phosphor-icons/react';

<Timeline>
  <TimelineItem>
    <TimelineDot tone="brand" icon={<Rocket weight="fill" />} />
    <TimelineContent>
      <TimelineTime dateTime="2026-05-28T14:02:00Z">há 5 min</TimelineTime>
      <TimelineTitle>Deploy em produção</TimelineTitle>
      <p>Versão v2.14.0 publicada.</p>
    </TimelineContent>
  </TimelineItem>
</Timeline>
```
