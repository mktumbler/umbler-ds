## Composição

```tsx
import { Textarea } from '@/components/ui/textarea';
import { InputGroup, InputLabel, InputHint } from '@/components/ui/input';

<InputGroup>
  <InputLabel htmlFor="bio">Biografia</InputLabel>
  <Textarea
    id="bio"
    rows={4}
    placeholder="Conte um pouco sobre você..."
    state="error"
  />
  <InputHint state="error">A biografia não pode estar vazia.</InputHint>
</InputGroup>
```

## API

| Prop          | Tipo                                    | Default     | Descrição                               |
| ------------- | --------------------------------------- | ----------- | --------------------------------------- |
| `state`       | `'default' \| 'error' \| 'success'`    | `'default'` | Estado de validação visual              |
| `rows`        | `number`                                | —           | Número de linhas visíveis               |
| `disabled`    | `boolean`                               | `false`     | Desabilita o campo                      |
| `...textarea` | `TextareaHTMLAttributes`                | —           | Todos os atributos nativos de textarea  |
