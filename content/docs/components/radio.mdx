## API

```tsx
import { Radio, RadioGroup } from '@/components/ui/radio';

<RadioGroup label="Plano" orientation="vertical">
  <Radio name="plan" value="starter" label="Starter" description="Para projetos pessoais." />
  <Radio name="plan" value="pro" label="Pro" description="Para agências e times." defaultChecked />
  <Radio name="plan" value="enterprise" label="Enterprise" disabled />
</RadioGroup>
```

| Prop (Radio)   | Tipo                    | Padrão      | Descrição                              |
| -------------- | ----------------------- | ----------- | -------------------------------------- |
| `label`        | `string`                | —           | Texto principal ao lado do radio.     |
| `description`  | `string`                | —           | Texto auxiliar abaixo do label.       |
| `disabled`     | `boolean`               | `false`     | Desabilita o radio.                   |
| `...input`     | `InputHTMLAttributes`   | —           | Atributos nativos de input.           |

| Prop (RadioGroup)  | Tipo                         | Padrão       | Descrição                              |
| ------------------ | ---------------------------- | ------------ | -------------------------------------- |
| `label`            | `string`                     | —            | Label do grupo (visualmente exposto). |
| `orientation`      | `'vertical' \| 'horizontal'` | `'vertical'` | Direção do layout dos radios.         |
