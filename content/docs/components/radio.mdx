---
title: Radio
description: Controle de seleção exclusiva em grupos, com orientação vertical ou horizontal.
---

Radio usa a API nativa `<input type="radio">` com `appearance-none` e um dot central animado via classes `peer-*` do Tailwind. Use `RadioGroup` para agrupar opções semanticamente.

## Composição

```tsx
import { Radio, RadioGroup } from '@/components/ui/radio';

<RadioGroup label="Plano" orientation="vertical">
  <Radio name="plan" value="starter" label="Starter" description="Para projetos pessoais." />
  <Radio name="plan" value="pro" label="Pro" description="Para agências e times." defaultChecked />
  <Radio name="plan" value="enterprise" label="Enterprise" disabled />
</RadioGroup>
```
