O `Badge` é usado para classificar ou indicar o status de um item. Use variantes semânticas (`success`, `error`, `warning`) para estados do sistema e `brand` ou `neutral` para categorização.

## Shapes

A prop `shape="tag"` troca a pílula arredondada por uma etiqueta retangular em caps + tracking. É o formato usado em changelogs, status de roadmap e release tags — leitura mais "log" do que "UI label".

## Uso

```tsx
import { Badge } from '@/components/ui/badge';

// Status
<Badge variant="success">Ativo</Badge>
<Badge variant="error">Cancelado</Badge>

// Com dot
<Badge variant="warning" dot>Em revisão</Badge>
```
