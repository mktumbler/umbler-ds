O `Badge` classifica, sinaliza estado ou rotula categorias. Use variantes semânticas (`success`, `error`, `warning`) para estados do sistema; `brand` para destaques; `neutral` para rótulos sem carga semântica. A variante `solid` é para badges flutuantes sobre superfícies coloridas.

## API

```tsx
import { Badge } from '@/components/ui/badge';

<Badge variant="success">Ativo</Badge>
<Badge variant="brand" dot>Online</Badge>
<Badge variant="warning" shape="tag">Beta</Badge>
<Badge variant="brand-solid" shape="pill">Mais popular</Badge>
```

| Prop      | Tipo                                                                                          | Padrão      |
| --------- | --------------------------------------------------------------------------------------------- | ----------- |
| `variant` | `brand \| success \| warning \| error \| neutral \| brand-solid \| success-solid \| warning-solid \| error-solid` | `neutral`   |
| `shape`   | `pill \| tag`                                                                                 | `pill`      |
| `dot`     | `boolean`                                                                                     | `false`     |
