Separa seções de conteúdo com uma linha discreta. Suporta orientação horizontal e vertical, além de um label central para divisores textuais.

## Variantes

<VariantGrid columns={1}>
  <VariantCard label="Horizontal / com label / vertical" hint="orientation='horizontal' (default), label central, orientation='vertical'">
    <SeparatorDemo />
  </VariantCard>
</VariantGrid>

## API

| Prop          | Tipo                           | Default        | Descrição                        |
| ------------- | ------------------------------ | -------------- | -------------------------------- |
| `orientation` | `'horizontal' \| 'vertical'`  | `'horizontal'` | Direção do separador             |
| `label`       | `string`                       | —              | Texto central (só horizontal)    |
| `className`   | `string`                       | —              | Classes extras                   |

## Uso

```tsx
import { Separator } from '@/components/ui/separator';

// Horizontal
<Separator />

// Com label
<Separator label="ou continue com" />

// Vertical (dentro de flex container com altura definida)
<div className="flex items-center h-8">
  <span>Esquerda</span>
  <Separator orientation="vertical" />
  <span>Direita</span>
</div>
```
