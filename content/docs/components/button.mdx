O `Button` cobre todas as ações do produto: confirmação, cancelamento, ações destrutivas e navegação. Sem dependência de Radix — React + Tailwind puro.

<Callout type="warn" title="Breaking change v0.x — pill default">
A partir desta versão, o Button tem **radius pill em todos os tamanhos** (decisão do [brand guide](/docs/marketing/brand)). CTAs primários em sizes `lg`/`xl` ganham `shadow-cta` (sombra azul brand). Quem instalou via shadcn antes da mudança precisa rodar `npx shadcn@latest add` de novo para receber a nova versão.
</Callout>

## Variantes

<ComponentPreview title="Variants">
  <ButtonVariantsDemo />
</ComponentPreview>

| Variante    | Uso                                      |
| ----------- | ---------------------------------------- |
| `primary`   | Ação principal da tela (CTA)             |
| `secondary` | Ação secundária, cancelamento            |
| `ghost`     | Ação terciária, toolbars, ícones inline  |
| `outline`   | Alternativa ao secondary em fundos claros |
| `danger`    | Ações destrutivas (excluir, revogar)     |

## Tamanhos

<ComponentPreview title="Sizes">
  <ButtonSizesDemo />
</ComponentPreview>

| Prop       | Altura | Padding H | Sombra (primary) | Uso típico                |
| ---------- | ------ | --------- | ---------------- | ------------------------- |
| `size="sm"` | 32px  | 12px      | `shadow-xs`      | Tabelas, inline           |
| `size="md"` | 40px  | 16px      | `shadow-xs`      | Padrão (default)          |
| `size="lg"` | 48px  | 24px      | `shadow-cta`     | Formulários, modais, CTAs |
| `size="xl"` | 56px  | 32px      | `shadow-cta`     | Hero de landing           |

Em landings/marketing, prefira `size="lg"` ou `size="xl"` — o `shadow-cta` (sombra azul brand) é o sinal premium do CTA. Em produto, `md` é o default funcional.

## Com ícones

Use os ícones do `@phosphor-icons/react`. Para botão só com ícone, passe `iconOnly` e um `aria-label`.

## Estados

<ComponentPreview title="States">
  <ButtonStatesDemo />
</ComponentPreview>

| Prop / Estado | Comportamento                                 |
| ------------- | --------------------------------------------- |
| `loading`     | Mostra spinner, desabilita o botão            |
| `disabled`    | Opacity 50%, pointer-events desativado        |

## Uso

```tsx
import { Button } from '@/components/ui/button';

// Básico
<Button>Salvar</Button>

// Variante e tamanho
<Button variant="danger" size="lg">Excluir conta</Button>

// Com loading
<Button loading>Salvando...</Button>

// Icon only
<Button variant="ghost" iconOnly aria-label="Configurações">
  <Gear size={20} />
</Button>
```

## Props

| Prop       | Tipo                                                  | Padrão      |
| ---------- | ----------------------------------------------------- | ----------- |
| `variant`  | `primary \| secondary \| ghost \| outline \| danger`  | `primary`   |
| `size`     | `sm \| md \| lg \| xl`                                | `md`        |
| `loading`  | `boolean`                                             | `false`     |
| `iconOnly` | `boolean`                                             | `false`     |
| `disabled` | `boolean`                                             | `false`     |

Aceita todos os atributos nativos de `<button>`.
