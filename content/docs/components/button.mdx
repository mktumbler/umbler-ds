O `Button` cobre toda ação clicável do produto: CTA, confirmação, cancelamento, ação destrutiva e navegação. Sem Radix — React + Tailwind puro.

<Callout type="warn" title="Breaking change v0.x — pill default">
A partir desta versão, o Button tem **radius pill em todos os tamanhos** (decisão do [brand guide](/docs/marketing/brand)). CTAs em `lg`/`xl` ganham `shadow-cta` (sombra azul brand). Quem instalou via shadcn antes precisa rodar `npx shadcn@latest add` novamente.
</Callout>

## Tamanhos

Em landing/marketing, prefira `lg` ou `xl` (a `shadow-cta` é o sinal premium). Em produto, `md` é o default funcional. `sm` é para densidade alta (tabelas).

## API

```tsx
import { Button } from '@/components/ui/button';

<Button>Salvar</Button>
<Button variant="danger" size="lg">Excluir conta</Button>
<Button loading>Salvando...</Button>
<Button variant="ghost" iconOnly aria-label="Configurações">
  <Gear size={20} />
</Button>
```

| Prop       | Tipo                                                  | Padrão      |
| ---------- | ----------------------------------------------------- | ----------- |
| `variant`  | `primary \| secondary \| ghost \| outline \| danger`  | `primary`   |
| `size`     | `sm \| md \| lg \| xl`                                | `md`        |
| `loading`  | `boolean`                                             | `false`     |
| `iconOnly` | `boolean`                                             | `false`     |
| `disabled` | `boolean`                                             | `false`     |

Aceita todos os atributos nativos de `<button>`.
