Use `DropdownMenu` para **ações secundárias** de um item (editar/duplicar/excluir), **opções rápidas** de configuração, ou **filtros**. Para seleção primária de valor em formulário, use `Select`.

## Anatomia

```tsx
<DropdownMenu>
  <DropdownMenuTrigger>...</DropdownMenuTrigger>
  <DropdownMenuContent>
    <DropdownMenuLabel>...</DropdownMenuLabel>
    <DropdownMenuItem>...</DropdownMenuItem>
    <DropdownMenuSeparator />
    <DropdownMenuCheckboxItem checked={x} onCheckedChange={setX}>...</DropdownMenuCheckboxItem>
    <DropdownMenuRadioGroup value={v} onValueChange={setV}>
      <DropdownMenuRadioItem value="a">...</DropdownMenuRadioItem>
    </DropdownMenuRadioGroup>
    <DropdownMenuItem destructive>Excluir</DropdownMenuItem>
  </DropdownMenuContent>
</DropdownMenu>
```

## Acessibilidade

- ↑↓ navegam itens, ↵ ativa, `Esc` fecha, `Tab` move foco para fora.
- Tipar uma letra pula para o próximo item que começa com ela.
- O trigger expõe `aria-haspopup="menu"` e `aria-expanded` automaticamente.

## Atalhos de teclado

`DropdownMenuShortcut` é apenas visual — não registra um listener global. Use `useHotkeys`/`react-hotkeys-hook` para o atalho funcionar de fato:

```tsx
<DropdownMenuItem onSelect={handleEdit}>
  Editar
  <DropdownMenuShortcut>⌘E</DropdownMenuShortcut>
</DropdownMenuItem>
```

## Uso

```tsx
import {
  DropdownMenu,
  DropdownMenuContent,
  DropdownMenuItem,
  DropdownMenuLabel,
  DropdownMenuSeparator,
  DropdownMenuTrigger,
} from '@/components/ui/dropdown';

<DropdownMenu>
  <DropdownMenuTrigger asChild>
    <Button>Ações</Button>
  </DropdownMenuTrigger>
  <DropdownMenuContent>
    <DropdownMenuLabel>Item</DropdownMenuLabel>
    <DropdownMenuItem onSelect={onEdit}>Editar</DropdownMenuItem>
    <DropdownMenuSeparator />
    <DropdownMenuItem destructive onSelect={onDelete}>
      Excluir
    </DropdownMenuItem>
  </DropdownMenuContent>
</DropdownMenu>
```
