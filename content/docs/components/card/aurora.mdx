`CardAurora` é uma variante de alto impacto para feature sections, bento grids e landing pages. O blob de fundo cria profundidade atmosférica sem depender de imagens externas.

## Uso

```tsx
import { CardAurora } from '@/components/ui/card-aurora';
import { Globe } from '@phosphor-icons/react/dist/ssr';

<CardAurora
  icon={<Globe size={22} />}
  label="Hospedagem"
  title="Infraestrutura global"
  description="Servidores no Brasil e no mundo."
  blob="sweep"
  rotation="default"
  className="w-[340px] h-[300px]"
/>
```

Para conteúdo personalizado passe `children` — o layout padrão (label/title/desc) é substituído:

```tsx
<CardAurora icon={<Globe size={22} />} blob="prism" className="h-[300px]">
  <div className="mt-6">
    {/* seu conteúdo aqui */}
  </div>
</CardAurora>
```

## Bento Grid

Combine com CSS Grid e `style={{ gridColumn: 'span 2' }}` / `style={{ gridRow: 'span 2' }}` para layouts bento:

```tsx
<div style={{ display: 'grid', gridTemplateColumns: 'repeat(3, 1fr)', gridTemplateRows: '260px 260px', gap: '16px' }}>
  <CardAurora ... style={{ gridColumn: 'span 2' }} />
  <CardAurora ... style={{ gridRow: 'span 2' }} />
  <CardAurora ... />
  <CardAurora ... />
</div>
```
