## API

```tsx
import {
  Card, CardHeader, CardTitle, CardDescription,
  CardContent, CardFooter,
} from '@/components/ui/card';

<Card variant="elevated" className="max-w-xs">
  <CardHeader>
    <CardTitle>Plano Pro</CardTitle>
    <CardDescription>Para agências que precisam de escala.</CardDescription>
  </CardHeader>
  <CardContent>
    <p className="font-heading text-h2 text-foreground tabular-nums">R$ 497</p>
  </CardContent>
  <CardFooter>
    <Button className="w-full">Assinar agora</Button>
  </CardFooter>
</Card>
```

| Prop       | Tipo                                              | Padrão      |
| ---------- | ------------------------------------------------- | ----------- |
| `variant`  | `default \| elevated \| ghost \| outline`         | `default`   |
| `className`| `string`                                          | —           |

| Sub-componente     | Padding padrão                         |
| ------------------ | -------------------------------------- |
| `CardHeader`       | `px-6 pt-6 pb-4`                       |
| `CardTitle`        | `text-body-lg font-semibold`           |
| `CardDescription`  | `text-body-sm text-foreground-muted`   |
| `CardContent`      | `px-6 py-4 flex-1`                     |
| `CardFooter`       | `px-6 pt-4 pb-6` com `border-t`        |
