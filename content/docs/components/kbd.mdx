Use `Kbd` para indicar uma tecla — `Esc`, `⌘`, `Tab`. Para combinações (`⌘ + K`, `Ctrl + Shift + P`), prefira `KbdSequence` que cuida do espaçamento e do separador.

## API — Kbd

| Prop        | Tipo                    | Default | Descrição                             |
| ----------- | ----------------------- | ------- | ------------------------------------- |
| `size`      | `'sm' \| 'md' \| 'lg'` | `'md'`  | Altura da tecla (20 / 24 / 28px)      |
| `children`  | `ReactNode`             | —       | Conteúdo — texto curto ou símbolo     |
| `className` | `string`                | —       | Classes extras                        |

## API — KbdSequence

| Prop        | Tipo                              | Default | Descrição                                                      |
| ----------- | --------------------------------- | ------- | -------------------------------------------------------------- |
| `keys`      | `ReactNode[]`                     | —       | Lista de teclas que compõem o atalho                           |
| `separator` | `'+' \| ' ' \| ReactNode`         | `' '`   | `' '` para combinação visual (⌘ K); `'+'` para soma (Ctrl + S) |
| `size`      | `'sm' \| 'md' \| 'lg'`           | `'md'`  | Propagado para cada Kbd                                        |
| `className` | `string`                          | —       | Classes extras                                                 |

## Uso

```tsx
import { Kbd, KbdSequence } from '@/components/ui/kbd';

// Tecla isolada
<Kbd>Esc</Kbd>

// Combinação macOS
<KbdSequence keys={['⌘', 'K']} />

// Combinação Windows/Linux
<KbdSequence keys={['Ctrl', 'Shift', 'P']} separator="+" />

// Atalho composto sequencial (estilo Vim/Gmail)
<KbdSequence keys={['G', 'I']} />
```
