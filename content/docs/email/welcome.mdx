Enviado logo após o cadastro. Apresenta o produto, reforça a marca e conduz o usuário
ao primeiro site com um CTA direto.

## Uso

```tsx
import { render } from '@react-email/render';
import WelcomeEmail from '@/emails/templates/welcome';

const html = await render(
  <WelcomeEmail
    firstName="Ana"
    ctaUrl="https://app.umbler.com/sites/new"
  />
);
```

## Estrutura do template

- **Badge** — "Conta criada" em azul
- **Título** personalizado com o nome do usuário
- **CTA principal** — "Criar meu primeiro site"
- **3 cartões de próximos passos** — domínio, publicação, e-mails
- **Nota de suporte** com link direto
