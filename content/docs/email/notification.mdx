Template transacional de alta confiança: exibe detalhes da fatura em um card estruturado,
inclui aviso de segurança (anti-phishing) e CTA direto para o painel de cobrança.

## Uso

```tsx
import { render } from '@react-email/render';
import NotificationEmail from '@/emails/templates/notification';

const html = await render(
  <NotificationEmail
    firstName="Ana"
    invoiceNumber="FAT-2025-0099"
    invoiceDate="01/06/2025"
    dueDate="10/06/2025"
    plan="Umbler Pro · 25 sites"
    amount="R$ 249,90"
    ctaUrl="https://app.umbler.com/billing/FAT-2025-0099"
  />
);
```

## Estrutura do template

- **Badge** — "Fatura emitida" em azul
- **Título** + parágrafo de contexto
- **Card de fatura** — valor em destaque, badge de status e tabela de detalhes
- **CTA** — "Ver e pagar fatura"
- **Aviso de segurança** — alerta anti-phishing em amarelo
- **Nota de suporte**
