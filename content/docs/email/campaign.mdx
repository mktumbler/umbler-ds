Template de campanha flexível: serve para anúncios de features, lançamentos de produto,
promoções sazonais e newsletters. O hero visual é parametrizável via props.

## Uso

```tsx
import { render } from '@react-email/render';
import CampaignEmail from '@/emails/templates/campaign';

const html = await render(
  <CampaignEmail
    firstName="Ana"
    featureTitle="Novo: Hospedagem Edge"
    featureDescription="Seu site entregue de 50+ pontos no mundo — latência abaixo de 50ms."
    ctaUrl="https://umbler.com/edge"
    ctaLabel="Conhecer hospedagem Edge"
  />
);
```

## Estrutura do template

- **Badge** — "Novidade" em azul
- **Hero box** — fundo gradiente com título e subtítulo parametrizáveis
- **Parágrafo de contexto** personalizado com o nome
- **CTA principal**
- **3 benefícios** com ícone + título + descrição em layout de duas colunas
