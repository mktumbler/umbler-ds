Transacional de segurança: CTA único de redefinição, expiração explícita, detalhes do
pedido (data + IP) para o usuário reconhecer a origem e aviso anti-phishing.

## Uso

```tsx
import { render } from '@react-email/render';
import PasswordResetEmail from '@/emails/templates/password-reset';

const html = await render(
  <PasswordResetEmail
    firstName="Ana"
    resetUrl={`https://app.umbler.com/reset?token=${token}`}
    expiresIn="1 hora"
    requestIp={ip}
    requestDate={formattedDate}
  />
);
```

## Estrutura do template

- **Badge** — "Segurança da conta"
- **Título** + parágrafo curto (foi você? então clique)
- **CTA** — "Criar nova senha" + expiração logo abaixo
- **Card de detalhes** — data/hora e IP do pedido (reconhecimento de origem)
- **Aviso "Não foi você?"** — instrução de ignorar + anti-phishing
- **Link em texto puro** — fallback pra quando o botão não renderiza
